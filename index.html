How It Works

1

Connect GitHub

Authorize Z-URL to access your repositories. Select the repo you want to deploy.

2

Configure Build

Choose your runtime, set the build command, start command, and add environment variables.

3

Deploy

Click deploy. We clone, build, and run your app. Get a *.zurl.app URL instantly.

4

Auto-Deploy

Every push to your branch triggers a new build and deploy. Automatic rollback on failure.

5

Scale Up

Upgrade your plan for more resources. Premium gets priority builds, more RAM, custom domains.

6

Monitor

Real-time logs, analytics, CPU/RAM metrics, and uptime monitoring from your dashboard.
